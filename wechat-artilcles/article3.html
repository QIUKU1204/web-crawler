P A R T. 1 

摩天大楼 

第一次到纽约就感受高度密集的摩天大楼无形的压力，它像被插满剑的靶子，密密麻麻。


相比较小短腿阿姆斯特丹（Amsterdam）的最高的建筑只有135m，曼岛真的是长腿欧巴了。

图片来源 http://skyscraperpage.com

「摩天大楼那些事儿」

1.新世贸中心是现在纽约市最高的建筑，高达541米。

2.在1890年前，纽约市最高的建筑是三一教堂（Trinity Church）直到New World Building（即Pulitzer Building） 建成。但是New world Building 1955年被拆除）。

3.克莱勒斯大厦（Chrysler Building）1930年完工，仅保持了11个月的世界最高建筑的记录就被超过。现在仍被视作全世界最高的砖造建筑物。

图片来源：Google

4.帝国大厦（The Empire State Building）曾经保持了42年（1931-1972）的世界最高建筑的记录。2001年9.11事件后，世贸双子塔倒塌，它再次成为了纽约市最高的建筑。


5.帝国大厦1930年开始建，花了1年多时间就建完了，速度贼快。并且造的时候对外报的虚报高度是300m，为了要赢克莱勒斯大厦，非常有心机地暗搓搓加到381m。


就像考完同桌问你，考的怎么样，你说一般般我超多不会做的。他说：我也是我也是，然后试卷发下来你96，他69的感觉。


图片来源：Google

6.19世纪初当大家都热情高涨，撸起袖子就要造摩天大楼。但是摩天大楼带来的光照问题也引发了很多争议，为了不让纽约的街道变成灰不溜秋的，1916年纽约市颁布第一部规范摩天大楼建造法案。

图片来源：Google

7. 第一个受到法案约束的摩天大楼是Barclay-Vesey Building (即现在的Verizon Building) 

*Barclay-Vesey Building: 498 英尺, 32 层，1923年开工，1927年建成。毗邻世贸中心，在9.11事件中也受到了不同程度的损毁。

图片来源：Google

「纽约、香港、上海、东京、首尔」

每个国家对于摩天大楼的建筑高度都有不同的定义。在美国普遍认为152m（500 ft）以上的高楼就算摩天大楼。而在中国当建筑高度超过100m，就都算超高层建筑。日本比较特殊，由于处于地震高频光顾区域，法律规定超过60m就是超高策层建筑。


纽约密集的建筑，让我联想到了香港，另外一个插满高楼大厦的城市。好奇心驱使下，自己统计了一下纽约、香港、上海、东京、首尔这5个城市的摩天大楼的数据。


纽约、香港、上海、东京、首尔 5个城市 

≥180m（已建成）的摩天大楼统计

「香 港」

香港现有118座180米以上的已建成的摩天大楼，118幢楼平均高218.11米，平均每幢建筑59.79层。环球贸易广场 （International Conference Center）是现在香港最高的建筑。

80、90年代是香港发展的黄金十年，也在这期间确立了香港国际金融中心的地位。政府推出高地价政策，香港楼市发展兴旺。香港经济逐步转型，由原来的轻工业、制造业（搭载改革开放的政策便利慢慢北移）慢慢转变成金融、旅游、地产等第三产业为支撑。

同时，随着人口的快速增加，香港有限的土地面积，需要利用超高大楼来谋求更多的生产空间。根据香港政府统计处的数据，从1981年到2000年，香港的人口从518万涨到666.5万人，人口增幅约25.9%。


1996年香港的人口Net movement达约11.9万,是1995年的18.64倍。大量人口净移入来自中国内地到香港是人口增长的主要原因。


Built and height ≥180m

在这样的背景下，香港的摩天大楼建造从80年代开始起，开始快速增长。非常著名的中银大厦（Bank of China Tower）就是在1989年建成的。


随着摩天大楼的建造技术成熟，和香港人地资源矛盾越来越突出，在21世纪的头10年就一发不可收拾，10年造78座180m以上的超高大楼分分钟冲出大气层，奔向月球。

图片来源：Google

头10年的用自己都怕的速度建了这么多，也差不多把这么小小的地方也填得差不多满了。于是2011-2017年香港低调做自己，建5座意思一下。


按照香港规划署提供的资料预测，香港本港实际可容纳823万人，加上有潜力的发展区域可容纳879万人。而截止到本文发稿时，香港政府统计处统计到的的人口已经有738.95万人。


还有1暂时还有100多万的剩余人口可容纳量，那满了的话，是不是要接着一边造摩天大楼，一边填海造陆呢？

图片来源：Google


「上 海」

上海现有74座180米以上的已建成的摩天大楼，74幢楼平均高231.189米，平均每幢建筑47.297层。上海塔（Shanghai Tower）是现在上海最高的建筑，暂排世界第二高。


上海比较多的超高建筑是在90年代开始建造，比如东方明珠塔1994年落成，在1994年至2007年间，是中国大陆地区的最高建筑。


2007年上海环球金融中心超越东方明珠塔。当然环球金融中心第一的宝座也没几年后就被现在的上海塔超越。

Built and height ≥180m

进入21世纪中国改革开放积累了大量的财富，有那个资本和技术去造更好的超级大楼，上海也没有辜负世界人民的期望，积极造摩天大楼（虽然没有香港造的那么凶）2001-2010年造的180m以上超高楼也是数量也是同时期曼岛的2倍了。


「东 京」

东京的已建成并且≥180m 的摩天大楼有44座，44幢楼平均高219.525米，平均每幢建筑44.522层。东京天空树（Tokyo Sky Tree）是东京都最高的建筑。


东京180m以上的摩天大楼从70年代开始迅速增加（当时日本也处于经济高速增长时期，地产事业蓬勃发展），甚至许多人相信“地产”是永远不会贬值的。除了本国地产的狂热投机，还疯狂进行海外地产投资。大量的投机活动导致了日本80年代后期-90年代初的经济泡沫大崩溃。


最有名的就是1989年10月三菱地所以8.46亿美圆（以当时的汇率，约合1100多亿日圆）的价格购买了洛克菲勒中心的拥有者洛克菲勒集团（Rockefeller Group）51%的股权（这被视为日本泡沫经济达到顶点）。当然后来1996年7月三菱地所因为无法偿还贷款破产，洛克菲勒集团将最初建造的14栋大楼转让。

built and height ≥180m

在进入20世纪后，随着造楼的技术的不断突破和资本的进一步累积，超高摩天大楼的建筑的建造也进入一个新的井喷时期。2011-2017年期间有明显的回落，毕竟日本本国是地震多发国家，摩天大楼对区位选择又比较挑，建造的数量也会慢慢减少也正常。


「 首 尔 」

首尔已建成并且≥180m 的摩天大楼有37座，乐天世界塔（Lotte World Tower）是首尔现在NO.1高的建筑。37座摩天大楼平均213.649m，平均52.216层。




从南山首尔塔望下去，也许是地势比较高，高楼大厦看上去都变得渺小，安静地地融入在夜景之中。


其实首尔超高摩天大楼的建造起步的时间晚于美国、日本等国家。相比东京在2011-2017年建造数量的回落，首尔还是再接再厉，一口气再造18个。


built and height ≥180m


「纽 约」

1.曼哈顿有117座180米以上的已建成的摩天大楼，117幢楼平均高223米，平均每幢建筑55.22层。

*数据由www.skycraperpage.com收集统计得到，截止到本文发稿时间。


摩天大楼的高度和数量也能侧面反应出一个地区、国家经济实力。纽约摩天大楼的建造开始非常早，19世纪初就开始蓬勃发展。在1961-1990年三个时间段里稳稳增长，不断造啊造。看来跟苏联冷战争霸的同时也没忘拼命造超级大楼。


Built and height ≥180m

当然拼命造高楼到达一点“饱和度”后，纽约歇了一歇（哈哈会是因为苏联解体，市区竞争动力了嘛？？）然后再接再厉，接着拆、改、造。在2001-2010年和2011-2017年这两个时间段发挥稳定。


「综 合 对 比」

5个城市180m以上已建成的摩天大楼建造对比图 


美国起步最早，日本后来在70-80年代也慢慢开始建摩天大楼，而中国香港、上海、首尔虽然起步晚，但是像香港这样高度密集的建摩天大楼也可以说是很有特色。

在美国，摩天大楼的建造跟私人资本和财团有着紧密的联系，纽约非常多的建筑直接以家族名称或者企业名称命名。比如：特朗普大厦（Trump Tower）,摩根大通大厦（JPMorgan Chase Tower）等等。日本也有新宿三井大厦和住友不动产新宿Oak大厦等以集团命名的。当然在上述统计的建筑里大部分是以所在地的地名、或者大厦的用途命名的。




帝国大厦在美国30年代经济大萧条时期的建成，又见证了美国的经济复苏。

由于当时建造条件有限，基本上没有防护措施的室外焊接，14个工人在建造过程中丧生。换来18个月不到就建成了这座传奇的建筑。

每座城市都有自己的气质，从夜景上就展示出巨大的差异。经典的俯瞰纽约的景点，哈德逊河连着Jersey city和Manhattan，建筑像巧克力一样方方正正被压缩在这个小小的岛上，纵横明亮的街道上经典的亮黄色的taxi慢慢的在狭窄的街道上移动。


图片来源：Google

先电梯到80楼了解一下帝国大厦的建造历史，在帝国大厦地基建好后，随后一年就建完了整幢楼。我当时心里一惊，随后跟小伙伴默默地放弃了电梯从80楼到86楼爬楼梯上去（有点小缺氧，会爬的比较累）。

Gossip Girl中毒，所以一定要上到86层的露台 86楼的露天观景台俯瞰整个曼岛，怀念一下Chuck和Queen B的故事。

图片来源：Google

86-102楼的区别：102楼是室内景观、就是高一点其他没差。纽约的夜景还是非常有特色的。曼岛的平均建筑密度和高度都在世界上首屈一指，高而密集。

出行攻略：请务必要提前订票，能省非常多的时间！！因为买票前需要排队安检一次，买票又要排长队、买完票进去参观又要排队。

上次提前订了票，但是可能因为长得比较VIP，所以莫名其妙被安排走了贵宾通道，也节省了很多时间。


「联合国总部」

纽约的联合国总部1952年建成，是第一个全部用玻璃装饰外部的摩天大楼，而外围蓝绿色的玻璃也是为了帮助更好吸收光点辐射，从而降低大楼内部的空调成本。


联大期间的安保还是满严格的，里面cafe的东西大部分都不太好吃，但是唯独有日式便当还不错！会议厅有很多，大部分配有同声传译器，前排标配小型电子屏。纪念品商店的东西挺贵的。 

联合国儿童基金会 妇女署都在周围的几个街区里。印象比较深刻的是在联大期间联合国总部外围的街区总有小规模的游行和抗议。想想杭州开G20的管控，这些游行是不敢想象的。

联合国总部也是可以预约参观的，如果感兴趣的朋友可以进去溜达一下。



「纽约微软中心」

纽约时报总部对面，有钱的科技公司能把办公环境布置的超有情调。露台上有调酒师提供饮料，还可以看来来往往的车流。还有就是准备的buffet实在太用心了。

拍摄于纽约微软技术中心的露台




「Hudson Yards」

Hudson Yards 也是欧莱雅、SAP 总部的所在地。在48层楼上俯瞰纽约，四周全部采用落地玻璃，在这样的环境的工作简直是享受。Coach的总部好像也在这边，里面有Coach包包历史的陈列柜（误打误撞看到的）。哈哈不知道Coach集团改名Tapestry转型后会发生什么有意思的事。




「后记」

摩天大楼象征财富、权利、对天空的渴望。他们勾勒出城市风格各异的天际线。
